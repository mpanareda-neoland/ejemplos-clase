Ejemplo Paginación

Creación de una sección con paginación

Combinaremos dos componentes de Bootstrap:

  • Paginación para la estética
  • Nav Toggleable para el comportamiento

Ejemplo básico

Usamos los dos componentes. Se puede observar que el ul tendrá las clases de ambos. No usaremos la clase .nav-pills o .nav-tabs en el ul para que no nos modifique a nive estético el componente.

Como el componente nav utiliza la clase .active en los a y el componente pagination lo espera en el li, podemos incluir un pequeño js que lo corrija.

Ejemplo dinámico

Si no sabemos de antemano cuántas páginas tendremos, en html pondremos sólamente los contenedores y desde jQuery generaremos el resto

Definiremos una función que reciba el número de la página y su contenido y la añada.

Al arrancar, iteramos el número de elementos que desemos crear y usamos la función para crearlas. Al finalizar, clickamos al primer elemento para que se cargue.

    Ejemplo avanzado

    Si lo que queremos es poner una conjunto de elementos en páginas, tendremos que calcular el total de páginas dependiendo del número de elementos por página que queramos

    Definiremos, como en el caso aterior, una función que reciba el número de la página y su contenido, ahora en formato json, y la añada.

    Al arrancar, leemos los elementos que queremos cargar, los agrupamos, e iteramos el número de páginas que desemos crear usando la función para crearlas. Al finalizar, clickamos al primer elemento para que se cargue.
