Ejercicios Javascript

Número de días del mes (Condicional)

Programa que pide que seleccionemos un mes y devuelve cuántos días tiene

Mayor de 3 números (Condicional)

Programa que pide 3 números y te dice cuál es menor

Lotería (Condicional)

Programa que te pide tu número y te dice si has ganado o si tienes reintegro (termina igual)

Contar las a (Bucle)

Programa que te pide una frase y te dice el número de letras a que tiene

Tabla de multiplicar (Bucle)

Programa que te pide un número y te devuelve la tabla de multiplicar de ese número

Adivina el número (Bucle)

Programa que pregunta cuántos intentos quieres tener para acertar y te pide un número esas veces para adivinar un número aleatorio
